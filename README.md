# Monte Carlo intersection of three circles

Файлы:

- `experiment.cpp` — программа для генерации экспериментальных данных (метод Монте-Карло).
- `data.csv` — сырые результаты экспериментов.
- `clean_data.xlsx` — те же данные в удобном для анализа виде (N, rect_type, area_est, rel_error).
- `area_vs_N.png` — график зависимости оценки площади от числа точек N.
- `rel_error_vs_N.png` — график зависимости относительной ошибки от N.

ID посылки в Codeforces (задача A1i): **347910118**.
